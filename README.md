# 📚 Library Management System

## Project Overview
A full-stack Library Management System built as a group project for CSE325 - 
Software Construction & Development.

## Tech Stack
- **Frontend:** React (Vite), React Router, Axios
- **Backend:** Node.js, Express.js
- **Database:** MySQL
- **Version Control:** Git & GitHub

## Team Members
- Member 1: Project Lead / Git Manager
- Member 2: Frontend Auth (Login/Register)
- Member 3: Frontend Book UI
- Member 4: Frontend CRUD Forms
- Member 5: Backend Auth API
- Member 6: Backend Books API

## How to Run

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run dev
```

## Git Workflow
- main branch → stable production code
- feature/* branches → individual features
- Pull requests required before merging
